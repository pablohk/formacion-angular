Validación runtime (Zod)

Al ser una librería JavaScript se puede usar en Anguar

Zod permite validar datos en tiempo de ejecución.

Existen otras librerías como Joi o Yup que también permiten realizar validaciones en tiempo de ejecución.

O usar las validaciones de formularios reactivos de angular (pero no podremos usarlas en las llamadas a APIs.) o usar libería de terceros, como Zod
